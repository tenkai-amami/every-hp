# every Inc. Website

株式会社everyの公式Webサイトのリポジトリです。

## 概要
法人向け貸施設「BASE」の運営会社、株式会社everyの公式サイトです。

## ページ構成
- `index.html` - トップページ
- `about.html` - 会社概要
- `service.html` - サービス紹介
- `contact.html` - お問い合わせ
- `success.html` - お問い合わせ完了ページ
- `assets/` - CSS、JavaScript、画像ファイル

## 技術スタック
- HTML5
- CSS3（レスポンシブデザイン対応）
- Vanilla JavaScript
